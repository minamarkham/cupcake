Sassy Starter

DEVELOPMENT

A starter toolkit based on Scalable and Modular Architecture for CSS (SMACSS) for Sass (SCSS) projects.

I wanted a Sass-based boilerplate for my websites, but didn't find one that utilitized my preferred CSS architecture, SMACSS. So I created my own. This is strictly a starter kit, not a framework, and is flexible enough for almost any project.
